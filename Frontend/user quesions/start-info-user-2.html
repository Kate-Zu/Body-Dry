Який ваш базовий рівень активності?

Виберіть те, що найкраще описує:

Не дуже активний
Проводьте більшу частину дня сидячи (наприклад, касир у банку, робота за столом).
Мало активний
Проводьте значну частину дня на ногах (наприклад, вчитель, продавець).
Активний
Проводьте значну частину дня, займаючись фізичною активністю (наприклад, офіціант, поштарь).
Дуже активний
Проводьте значну частину дня, займаючись важкою фізичною активністю (наприклад, кур'єр на велосипеді, тесляр).
